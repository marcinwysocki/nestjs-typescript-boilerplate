A <img src="https://github.com/remojansen/logo.ts/blob/master/ts.png?raw=true" width="18"> boilerplate for [Nest](https://github.com/kamilmysliwiec/nest), including 
nodemon, TSLint 
and Prettier (for now).

## Installation

```
$ npm install
```

## Start

```
$ npm run start
```
